# LineFollowerBot
Github Repository for Line follower Robot

Aim is to create a fast Line follower bot, using PID algorithm.
Components used: 
1. Raspberry Pico 
2. TB6612FNG motor driver
3. N20 motors 600rpm
4. Cytron IR array 
5. 7805 voltage regulatory
6. 7.4v 850mah lipo battery.
7. Custom made PCB.
